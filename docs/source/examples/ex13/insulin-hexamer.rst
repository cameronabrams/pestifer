Example 13: DES-PHE B1 Bovine Insulin Hexamer
---------------------------------------------

`PDB ID: 2ins <https://www.rcsb.org/structure/2ins>`_ is a structure of the bovine insulin hexamer with the DES-PHE B1 mutation.  This example demonstrates how to build a system with a hexameric assembly, and it also illustrates how pestifer automatically handles rare bonds, like the HIS-NE-Zn bonds.
