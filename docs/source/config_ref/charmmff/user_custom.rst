``user_custom``
===============

This attribute allows you to specify user-defined custom charmmff files that are not included in the standard CHARMMFF release.  You can use this to provide your own custom files for use with pestifer.


Example:
++++++++

.. code-block:: yaml

   charmff:
     user_custom:
       searchpath:
       - /path/to/my/custom/search/path/
       rtf:
       - a_file.rtf
       str:
       - a_file.str
       prm:
       - a_file.prm
