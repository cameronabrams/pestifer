``segtypes``
------------



Optional per-resname segtype overrides for residues defined in user_custom files. Each key is a segtype (e.g. 'ligand', 'cofactor') and each value is a list of resnames to classify under that segtype. Resnames not listed here are classified as 'ligand' by default.
