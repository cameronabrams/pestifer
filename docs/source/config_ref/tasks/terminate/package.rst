``package``
===========

Task that packages all CHARMM parameter files and pdb/psf/coor/vel/xsc files for a production run

Single-valued parameters:

  * ``state_dir``: directory name prepended to all files in the final state tarball

  * ``basename``: Desired basename for final tarball and the files within

  * ``ensemble``: Name of ensemble to run in; note that NPT can only be performed on a solvated system

  * ``nminsteps``: Number of minimization steps

  * ``nsteps``: Number of MD time steps

  * ``firsttimestep``: First timestep to use in the run (if unset, inherits from last MD task)

  * ``dcdfreq``: number of time steps between configuration output to DCD file

  * ``xstfreq``: number of time steps between cell size output to DCD file

  * ``restartfreq``: number of timesteps between restart checkpoints

  * ``temperature``: Temperature (K) for NVT or NPT

  * ``pressure``: Pressure (bar) for NPT; note that NPT can only be performed on a solvated system

  * ``outputenergies``: timesteps between outputting energies to the log file

  * ``other_parameters``: key:value pairs for other namd3 configuration file statements
