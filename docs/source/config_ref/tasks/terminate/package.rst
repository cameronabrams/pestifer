``package``
===========

Task that packages all CHARMM parameter files and pdb/psf/coor/vel/xsc files for a production run

Single-valued attributes:

  * ``state_dir``: directory name prepended to all files in the final state tarball

  * ``basename``: Desired basename for final tarball and the files within
