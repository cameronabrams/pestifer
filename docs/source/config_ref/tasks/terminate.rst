``terminate``
=============

Task that signals end of task list

Single-valued parameters:

  * ``basename``: Desired basename for final psf/pdb files (default: my_system)

  * ``chainmapfile``: Desired name of the file describing all chain mapping (default: chainmaps.yaml)

  * ``statefile``: Desired name of the file describing the current state (default: states.yaml)
