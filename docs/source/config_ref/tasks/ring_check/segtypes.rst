``segtypes``
------------

  * lipid
  * glycan


list of segment types whose rings are checked for piercing. Add 'protein' to also check side-chain rings (His/Phe/Tyr/Trp/Pro); a pierced protein ring is resolved by rotating the side-chain dihedral (chi2, then chi1) rather than by deletion
