``mods``
========

Specifies modifications to the sequence or structure in the current structure

Single-valued attributes:

  * ``irotations``: internal-coordinate dihedral rotations to reposition residues

  * ``crotations``: alias for irotations (internal-coordinate dihedral rotations)

  * ``transrot``: global translations and rotations

  * ``rottrans``: alias for transrot (global translations and rotations)

  * ``orient``: orient structure along principal axes

  * ``align``: rigidly align the pipeline system to a reference coordinate file

  * ``transfer_coords``: copy atomic coordinates from a donor system onto the pipeline system
