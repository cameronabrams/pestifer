``solvate``
===========

Parameters controlling solvation and ionization

Single-valued attributes:

  * ``solvent``: solvent species (RESI name); TIP3/water uses VMD's built-in pre-equilibrated water box, any other name is looked up as a kind:box entry in the 'solvent' PDB collection and tiled via solvate -spsf/-spdb/-ws/-ks (default: TIP3)

  * ``pad``: padding value for determining the overall simulation box size (in Angstroms) (default: 10)

  * ``salt_con``: salt concentration (M); argument for -sc for autoionize

  * ``neutralize``: add counter-ions to cancel the system's net charge so the built system is neutral. Enabled by default. For water this is autoionize's usual neutralization; for a non-water solvent it replaces solvent molecules with ions. Set false to keep the system's net charge (NAMD neutralizes with a uniform background under PME) (default: True)

  * ``cation``: name of cation

  * ``anion``: name of anion
