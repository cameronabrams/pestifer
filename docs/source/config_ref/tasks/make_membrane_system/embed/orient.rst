``orient``
==========

general vector-based orientation; rotates the protein so that a source vector is carried onto a target vector (the same ALIGN operation as the manipulate task's transrot mod).  Takes precedence over z_head_group/z_tail_group when given.  Each of source/target is either a literal 3-vector [x,y,z] or a pair of atomselections [selA, selB] whose mass-weighted centers define the vector (from selA to selB).

Single-valued attributes:

  * ``source``: source vector to be reoriented; a literal [x,y,z] or a pair of atomselections [selA, selB]

  * ``target``: target vector onto which the source is carried (e.g. the membrane normal [0,0,1]); a literal [x,y,z] or a pair of atomselections [selA, selB]
