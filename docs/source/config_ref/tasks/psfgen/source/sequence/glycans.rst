``glycans``
===========

parameters controlling how model-built glycans are adjusted, if present

Single-valued attributes:

  * ``numbering``: Glycan resid numbering convention: 'narrow' = block-allocate above max protein resid; 'wide' = root monomer gets protein-attachment resid + wide_shift, subsequent monomers increment by 1 (default: narrow)

  * ``wide_shift``: Resid offset added to the protein attachment resid for the 'wide' glycan numbering convention (e.g. ASN 123 → NAG 3123 with default shift of 3000) (default: 3000)

  * ``max_glycan_size``: Maximum number of residues per glycan tree; used to block-allocate unique resid ranges in the 'narrow' convention so glycans can be extended later without resid conflicts (default: 30)
