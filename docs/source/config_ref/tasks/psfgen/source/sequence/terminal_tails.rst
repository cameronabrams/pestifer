``terminal_tails``
==================

Grouped control for building unresolved N-/C-terminal residue runs ("tails"). Interior gaps are always built and closed (see the loops block); terminal tails are opt-in here and, when built, modeled by the free-tail modeler (loops.declash.model_tails). Replaces the flat build_zero_occupancy_N_termini / build_zero_occupancy_C_termini / include_terminal_loops keys below, which are still accepted (with a one-time deprecation warning) and folded into this block.

Single-valued attribute:

  * ``all``: Build both N- and C-terminal unresolved runs on every chain (equivalent to the legacy include_terminal_loops) (default: False)
