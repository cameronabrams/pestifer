``loops``
=========

Parameters controlling model-built loops during psfgen

Single-valued parameters:

  * ``sac_res_name``: 3-letter code of residue appended to any sufficiently long model-built (default: GLY)

  * ``min_loop_length``: Minimum length of model-built loop that triggers the ligation algorithm (default: 4)
