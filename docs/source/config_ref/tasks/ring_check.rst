``ring_check``
==============

Parameters controlling pierced-ring checks

Single-valued parameters:

  * ``delete``: which of the piercer or piercee to delete; if none, no deletions are performed (default: piercee)

  * ``cutoff``: distance cutoff between bond and ring COM's for checking, Angstroms (default: 10.0)
