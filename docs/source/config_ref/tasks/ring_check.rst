``ring_check``
==============

Parameters controlling pierced-ring checks

Single-valued attributes:

  * ``delete``: which of the piercer or piercee to delete; if none, no deletions are performed (default: piercee)

  * ``cutoff``: distance cutoff between bond and ring COM's for checking, Angstroms (default: 10.0)

  * ``max_ring_size``: only rings of at most this many atoms are considered (keeps real 5/6-membered rings while skipping the giant chordless cycles a disulfide makes through the protein backbone) (default: 7)
