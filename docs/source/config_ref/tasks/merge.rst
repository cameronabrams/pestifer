``merge``
=========

Merge two or more pre-built PSF/PDB systems into a single PSF/PDB system

Single-valued attributes:

  * ``collision_strategy``: how to resolve segment name collisions across systems; ``enumerate`` appends a numeric suffix to the conflicting name, ``error`` raises immediately on any unresolved collision (default: enumerate)

List-valued attribute:

  * ``systems``: ordered list of systems to merge; each entry must supply ``psf`` and either ``pdb`` or ``coor``, and may optionally supply ``segname_map`` (a dict of ``{old_segname: new_segname}`` explicit renames for that system)
