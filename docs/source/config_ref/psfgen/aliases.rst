``aliases``
===========

Add any residue or atom aliases needed; this is often done to make sure you have a 4-byte resname for any residue.


Example:
++++++++

.. code-block:: yaml

   aliases:
     atom:
     - ILE CD1 CD
     residue:
     - HIS HSD
